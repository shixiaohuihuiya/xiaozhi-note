# 小智笔记 (XiaoZhi Notes) - 产品需求文档

---

## 1. 项目概述

### 1.1 产品名称
小智笔记 (XiaoZhi Notes)

### 1.2 产品定位
一款集成AI智能辅助的个人笔记管理系统，支持用户撰写、管理和分享笔记，同时提供豆包AI扩展功能，帮助用户在写作过程中获得智能建议和补充。

### 1.3 目标用户
- 个人知识管理用户
- 博客作者
- 内容创作者
- 学生和学习者

## 3. 系统架构

### 3.1 前端架构
- **框架**: Vue 3 
- **UI组件库**:  Ant Design
- **状态管理**:  Redux
- **路由**: Vue Router
- **HTTP客户端**: Axios
- **编辑器**:  Markdown-it

### 3.2 后端架构
- **框架**: FastAPI (Python 3.11+)
- **数据库**: MySQL 8.0 
- **ORM**: SQLAlchemy 2.0
- **缓存**: Redis
- **搜索引擎**: Elasticsearch (可选)
- **文件存储**: MinIO 
- **AI服务**: 豆包API
- **认证**: JWT (python-jose)
- **文档**: FastAPI 自动生成 OpenAPI/Swagger

### 3.3 部署架构
- **容器化**: Docker
- **反向代理**: Nginx
- **CI/CD**: GitHub Actions / Jenkins

## 5. 非功能性需求

### 5.1 性能需求
- 页面加载时间 < 2秒
- API响应时间 < 500ms
- 支持并发用户 1000+

### 5.2 安全需求
- 密码加密存储（BCrypt）
- JWT Token认证
- SQL注入防护
- XSS攻击防护
- CSRF防护
- 接口限流

### 5.3 可用性需求
- 系统可用性 99.9%
- 数据备份机制
- 故障恢复机制
