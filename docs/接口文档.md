# 小智笔记 - API接口文档

---

## 1. 接口规范

### 1.1 基础信息

- **协议**: HTTPS
- **数据格式**: JSON
- **字符编码**: UTF-8
- **Content-Type**: `application/json`

### 1.2 接口地址

```
开发环境: http://localhost:8000
生产环境: https://api.xiaozhi-notes.com
```

### 1.3 响应格式

```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 1.4 状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 401 | 未授权/Token失效 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 422 | 验证错误 |
| 429 | 请求过于频繁 |
| 500 | 服务器内部错误 |

### 1.5 认证方式

使用 JWT Bearer Token 认证：

```
Authorization: Bearer <access_token>
```

## 2. 认证模块 (Auth)

### 2.1 用户注册

**接口**: `POST /api/v1/auth/register`

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| username | string | 是 | 用户名，3-50字符 |
| email | string | 是 | 邮箱地址 |
| password | string | 是 | 密码，6-32字符 |
| captcha | string | 是 | 验证码 |

**请求示例:**

```json
{
  "username": "zhangsan",
  "email": "zhangsan@example.com",
  "password": "123456",
  "captcha": "a3f8"
}
```

**响应示例:**

```json
{
  "code": 200,
  "message": "注册成功",
  "data": {
    "user_id": 1,
    "username": "zhangsan",
    "email": "zhangsan@example.com"
  }
}
```

---

### 2.2 用户登录

**接口**: `POST /api/v1/auth/login`

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| account | string | 是 | 用户名或邮箱 |
| password | string | 是 | 密码 |

**请求示例:**

```json
{
  "account": "zhangsan",
  "password": "123456"
}
```

**响应示例:**

```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIs...",
    "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
    "token_type": "Bearer",
    "expires_in": 3600,
    "user": {
      "id": 1,
      "username": "zhangsan",
      "nickname": "张三",
      "avatar": "https://...",
      "role": 0
    }
  }
}
```

---

### 2.3 刷新Token

**接口**: `POST /api/v1/auth/refresh`

**请求头:**

```
Authorization: Bearer <refresh_token>
```

**响应示例:**

```json

  "data": {
```

### 4.3 创建文章

**接口**: `POST /api/v1/articles`

**请求参数:**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | string | 是 | 文章标题，2-200字符 |
| content | string | 是 | 文章内容 |
| content_type | int | 否 | 1-Markdown, 2-HTML，默认1 |
| summary | string | 否 | 文章摘要 |
| cover_image | string | 否 | 封面图片URL |
| category_id | int | 否 | 分类ID |
| tag_ids | array | 否 | 标签ID数组 |
| status | int | 否 | 0-草稿, 1-发布，默认0 |
| is_top | boolean | 否 | 是否置顶，默认false |

```json
{
  "title": "我的第一篇文章",
  "content": "# 标题\n\n正文内容...",
  "content_type": 1,
  "summary": "这是文章摘要",
  "cover_image": "https://example.com/cover.jpg",
  "category_id": 1,
  "tag_ids": [1, 2, 3],
  "status": 1
}
```

**响应示例:**

```json
{
  "code": 200,
  "message": "创建成功",
  "data": {
    "id": 1,
    "title": "我的第一篇文章",
    "slug": "my-first-article",
    "status": 1,

```
